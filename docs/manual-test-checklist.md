# focusbm GUI 手動検証チェックリスト

XCTest では担保できない View 描画・NSEvent 経路・アニメーションを、人間の目視と操作で確認するためのチェックリストです。

## Overview

- 対象環境: macOS GUI 起動
- 設定ファイル: `~/.config/focusbm/bookmarks.yml`
- 推奨手順: 1 項目ずつ設定を切り替え、アプリを再起動して結果を確認
